Limt

A free to use web service that enables users to upload files and generate keys to allow controlled access to these files. Initially created to facilitate sharing our files selectively with our clients and provide alerts when the file keys were consumed, we eventually decided to give access to the general public due to popular demand. Built on a heavily modified version of Ruby on Rails.
